# libconsole
✨ Beautiful printouts in Python

# Usage

Install:

```
pip install libconsole
```

Example:

```python
>>> import libconsole
>>> libconsole.print_box(rows=["Hello"])
╭─────────╮
│  Hello  │
╰─────────╯
```
