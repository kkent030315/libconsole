# libconsole
✨ Beautiful printouts in Python

https://pypi.org/project/libconsole/

# Usage

Install:

```
pip install libconsole
```

Example:

```python
>>> import libconsole
>>> libconsole.print_box(rows=["Hello"])
╭─────────╮
│  Hello  │
╰─────────╯
```
